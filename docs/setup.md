# Setup

1. Make sure that you have installed [uv](https://docs.astral.sh/uv/)
2. Clone project
    
    ```bash
    git clone git@github.com:morphin77/DualBraneCosmology.git
   ```
3. Install dependencies

    ```bash
    uv sync
   ```

4. To run example execute in project root
    
    ```bash
   uv run main.py
   ```
